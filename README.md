# RSS Reader

複数のRSSフィードをタブで管理できるWebアプリケーションです。

## 技術スタック

- Next.js 15 + React 19 + TypeScript
- Tailwind CSS v4
- shadcn/ui

## セットアップ

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認できます。

## 機能

- RSSフィードの追加・削除
- タブによる複数フィード管理
- レスポンシブデザイン
- ダークモード対応

## 開発コマンド

```bash
npm run dev    # 開発サーバー
npm run build  # ビルド
npm run lint   # リント
```
